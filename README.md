## üí° Features:

- Quick Code Checking : CookOff uses Judge0 as its compiler, which allows for fast and accurate code checking. Users get immediate feedback on whether their code is correct and how well it performs.
- Variety of Problems : CookOff has many programming problems for all skill levels, from beginners to experts. The problems cover different programming languages and data structures, so there‚Äôs something for everyone.
- Admin Dashboard : For platform administrators, there‚Äôs a user-friendly dashboard that helps manage user accounts, monitor system performance, and control user-generated content.
- Timer : The countdown timer shows users how much time is left for their coding sessions or contests. It clearly displays the remaining time, helping users manage their time effectively.
- Multiple Coding Languages : CookOff offers a wide range of programming languages for users to choose from when solving problems. Whether you prefer Python, Java, C++, JavaScript, or others, you can select the language that suits you best.
- Score Dashboard : The score dashboard gives users a clear view of their performance in various rounds of contests. It shows scores for each round, allowing users to track their progress over time.

## üèÅ Get Started

To get started, fork your own copy and clone the master branch. To clone a branch you can run the following:

```bash
git clone -b master https://github.com/<Your username>/cookoff-portal-9.0.git
```

Run these commands on your bash/terminal and open it in a code editor of your choice.

Run the following to install all the dependencies:

```bash
pnpm i
```

To start your development server run:

```bash
pnpm dev
```
